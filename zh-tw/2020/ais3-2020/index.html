Reverse

FallenBeat

題目是一個 jar 檔案，使用 JD-GUI 就可以反編譯了

首先先在 PanelEnding.class 內找到 flag

接下來往下摸可以看到 flag 透過跟 cache 做 xor 之後就是我們要的 flag

往上摸看到 cache 是參數傳進來的，使用全域搜尋找到 GameControl.class 使用了這個函數

跟著看可以看到他是存某個檔每一行的數字，且檔名是參數傳進來的，再次用全域搜尋找到 Frame.class 使用了這個函數，可以看到檔案為 songs/gekkou/hell.txt

接下來就是做一串處理就可以得到 flag 了

Web

Elephant

在 CTFd 上面有提示網頁有提示，開 devtool 就可以看到了

看到提示 Source Code 就會想到 Git，但是回傳 Forbidden，賭他規則沒寫好，瀏覽 /.git/HEAD 就會發現他只有資料夾，檔案都沒擋到

透過 GitHack 可以將原始碼下載下來

可以看到我們基本上是要做到 canReadFlag() 這個函式要回傳 1。
這個時候 php 利用優秀的特性就可以做到了，當 strcmp 兩個型態不相同時會回傳 0，所以你可以改成 NULL 或是 array。

在自己的機器上寫出相同的 Code ，只是將 $this->token 改成 NULL即可。
