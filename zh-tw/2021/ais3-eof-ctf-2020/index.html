AIS3 EOF CTF 2020 心得

Created2021-02-11Updated2021-02-11

Qual

在組隊的時候就差點找不到隊友(其實是我一直忘記這件事)，最後從 NCtfU 中直接拉滿人。
這場最刺激的其實不是比賽，是比賽剛好在期末考前一週的週末，由於已經組好對了，所以就抱著期末考之後熬夜看就好的心態去比賽 XDDDD

初賽的部分其實貢獻不多，大多都是跟別人討論完題目之後別人就解出來了(隊友超級 Carry Orz)，所以其實我初賽可以說貢獻值 == 0 QQ

最後跑去解 Abexcm100，一題 terrynini 出的 reverse，最後發現是我想太難了。
一開始檔案拉進 PEiD 發現是 PEspin Packed 的時候就打算直接硬拆他還一直找 OEP。
結果最後發現其實題目要的只是你慢慢追程式的行為而已，在初賽結束後半個小時才把他解掉了

初賽最後拿到了 12/15 晉級決賽！

Final

比賽前一天還跑去打工，結果隔天累爆QQ。
第一天 9:30 開始報到，結果我 9:15 就到了，路上還遇到蔡粽大佬 Orz，開賽前也剛好遇到俊穎教授還有欣明教授，寒暄幾句之後就開始比賽惹。

一開賽就喵了一眼 Web 題分別是 Regex Master 跟 WhoisTool 發現分別是 JavaScript 跟 Golang 寫了題目之後果斷放棄(欸，跑去看 Reverse。

Reverse 大部分都是 ARAY 系列題，是 Inndy 大佬出的，開來看之後發現是 Yara Rule，其中 rule 是經過 yarac 編譯過的。

身為一個一言不合就硬幹的玩家，看到編譯過的檔案就是先用 strings 看，結果就猜到了 XD。
檔案裡面有 44 個 Flag (只有其中一個是對的)，一開始就無腦挑幾個送就中了 XDD。後來回去看題目敘述才發現有提示可以把 Flag 各自拆成檔案，這才想起來應該要用 yara 下去掃才對 (這才是正規解)

ARAY2 的題目就基本上是一段一段的 flag 用變數存起來，只是我是用 hexdump -C 所以 output 會是 Hex + ASCII，看起來超醜，花了一點時間才拼出 flag。

第一天的進度大概就這樣，看完 ARAY 前面兩題之後就又跑回去看 Web，雖然沒有任何進展 XDD，最後好像 9 點多就跳去床上爆睡了(還被地震搖醒)，一路睡到隔天早上。

第二天大概就是第一天後半段的無限輪迴，不斷對著 Code 發呆，好險在第二天的下午合力解出了兩題，要不然真的會墊底 QQ。

最後名次 8/25 (包含 GCC)

其實 Final 還有一件趣事，在前些時間 ISIP 還有跟其他國家共同舉辦 GCC，因為剛好認識的教授有被邀請去當講師，教授就指派我幫忙出題目，其中有兩題是原本打算 GCC 中最後的 Final 拿出來，結果後面好像欣明教授還有其他教授決定 GCC Final 跟 EOF Final 合辦，所以原本我出的題目也會出現在 EOF Final XDD。

最後

感謝隊友的無敵 Carry，在最後面的 Final 我們拿到了潛力獎，近期都在用實驗室的東西，已經跟 CTF 有點脫節了，最後還有拿獎其實有點意外 XD。

感謝我的隊友 r888800009, maple3142, JyunD 帶我飛 Orz
